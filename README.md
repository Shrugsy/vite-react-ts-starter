# Vite/React/Typescript Starter

A starter repository with the following libraries set up and included:

- Vite
- React
- Typescript
- Eslint
- Prettier
- @emotion/react
- @emotion/styled
- msw
- @reduxjs/toolkit
- react-redux
- @mui/material
- jest
- @testing-library/react

## Usage

### Run in development mode

Running in development mode will re-compile and hot-reload on changes.

```bash
npm run dev
```

### Run in production mode

```bash
npm run build
npm run serve
```

### Run tests

```bash
npm run test
```

In watch mode:

```bash
npm run test-watch
```

### Run type checking

```bash
npm run typecheck
```

### Run linting

```
npm run lint
```
